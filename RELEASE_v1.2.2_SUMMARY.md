# Hunter v1.2.2 — Advanced UI Restoration

## 🎯 Release Summary

### New Features
- **🧭 Restored Advanced workspace** — Stats, Configs, Logs, Docs, and Runtime tools are now consolidated into a dedicated `Advanced` page
- **📊 Live discovery counters** — The simplified dashboard now shows `Available now` and `Checked` counts in real time

### Bug Fixes
- **🧩 Advanced tab stability** — Tab selection no longer resets during routine rebuilds or live status refreshes
- **🗂️ Correct advanced routing** — Legacy navigation to Statistics, Logs, and Docs now opens the matching tab inside `Advanced`

### UI Improvements
- **♻️ Continuous-cycle messaging** — The dashboard clearly explains that Hunter repeats discovery passes continuously
- **🧼 Cleaner main experience** — Technical tools remain fully available without cluttering the simplified dashboard

---

## 🚀 Download

- **Windows**: `hunter_dashboard.exe` + `hunter_cli.exe` + `HunterVPN-Setup-v1.2.2.exe`
- **Requirements**: Windows 10+, minimum 4GB RAM

## 📝 Quick Start

1. Run `hunter_dashboard.exe`
2. Click **START HUNT** to begin discovery
3. Track live counts on the dashboard
4. Open **Advanced** for logs, docs, metrics, and runtime controls
