## 🚀 Installation

### Prerequisites
- Windows 10/11 x64
- 4GB+ RAM recommended
- Internet connection for config discovery

### Steps
1. Download `Hunter-v1.0.0-Final.zip` (64.55 MB)
2. Extract to any directory (e.g., `C:\Hunter\`)
3. Run `hunter_dashboard.exe`
4. Configure Telegram settings in UI (optional)
5. Click START to begin config discovery

**Note**: No installation required. Fully portable and self-contained.

- Raw configs: `config/All_Configs_Sub.txt`, `config/sub.txt`

- Settings: `runtime/hunter_config.json`

```

│   ├── xray.exe

│   ├── mihomo-windows-amd64-compatible.exe
```

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| hunter_cli.exe not found | Ensure bin/ directory exists with hunter_cli.exe |
| DLL errors | All DLLs should be bundled; check extraction was complete |
| Port 10808 in use | Change port in runtime/hunter_config.json |
| No configs found | Check internet connection and Telegram settings |
