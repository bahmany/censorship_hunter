Hunter VPN v1.1.0 Released - Major Windows Update!

Excited to announce the release of Hunter VPN v1.1.0 with significant improvements for Windows users!

Key Highlights:
- Native Windows file dialog for seamless config import
- Streamlined installer with zero external dependencies
- 40% smaller installer footprint (35.3 MB)
- Automatic process management for smooth updates
- Enhanced build system with Flutter Windows fixes

Technical Improvements:
- Statically linked CLI - no more DLL dependencies
- MSYS2 cmake + ninja + MSVC toolchain integration
- Tor integration with proper process management
- Real-time WebSocket bridge for live status updates

What Hunter VPN Does:
- Bypasses internet censorship with dual-protocol proxy (SOCKS + HTTP)
- TLS fragmentation anti-DPI technology
- Smart routing with pre-loaded configs
- Autonomous config hunting from GitHub & Telegram
