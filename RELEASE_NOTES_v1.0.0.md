# 🚀 Hunter v1.0.0

> **Autonomous Proxy Discovery & Load Balancing Engine**

---

## ✨ Why Hunter?

**Hunter automatically finds, tests, and load-balances the best proxy configurations from 20+ sources.**

- 🔥 **Zero Configuration** — Just run and start
- ⚡ **Fast & Powerful** — C++17 backend with 9 parallel threads
- 🛡️ **Censorship Resistant** — Advanced DPI evasion strategies
- 📊 **Beautiful Dashboard** — Real-time monitoring with Flutter
- 🔄 **Fully Autonomous** — Works without any intervention

---

## 🎯 What It Does?

1. **Discover** — Collects configs from GitHub and Telegram channels
2. **Test** — Measures ping and quality with XRay
3. **Classify** — Gold (<2s), Silver (<5s), or Dead
4. **Load Balance** — Serves best configs on port 10808
5. **Monitor** — Continuously checks config health

---

## 🚀 Quick Start

1. **Download** `Hunter-v1.0.0-Final.zip`
2. **Extract** anywhere (no installation needed)
3. **Run** `hunter_dashboard.exe`
4. **Click START** — Automatic discovery begins
5. **Set browser** to `127.0.0.1:10808` (SOCKS5)

That's it! The system automatically finds and prepares the best configs.

### 🔄 Smart Learning
- **Memory** — Remembers good configs
- **Continuous Testing** — Always checking health
- **Optimization** — Selects the best options

> **موتور خودکار کشف و بارگذاری متعادل‌کننده پروکسی**

---

## ✨ چرا Hunter؟

**Hunter به طور خودکار بهترین کانفیگ‌های پروکسی را از 20+ منبع پیدا می‌کند، تست می‌کند و برای شما آماده می‌کند.**

- 🔥 **صفر تنظیمات** — فقط اجرا کنید و شروع کنید
- ⚡ **سریع و قدرتمند** — بک‌اند C++17 با 9 نخ موازی
- 🛡️ **مقاوم سانسور** — استراتژی‌های پیشرفته دور زدن DPI
- 📊 **داشبورد زیبا** — مانیتورینگ زنده با Flutter
- 🔄 **خودکار** — کار می‌کند بدون نیاز به دخالت

---

## 🎯 چه کار می‌کند؟

1. **کشف** — از GitHub و کانال‌های تلگرام کانفیگ جمع‌آوری می‌کند
2. **تست** — با XRay پینگ و کیفیت را اندازه‌گیری می‌کند  
3. **طبقه‌بندی** — طلا (<2ثانیه)، نقره (<5ثانیه)، یا مرده
4. **بارگذاری متعادل** — بهترین‌ها را روی پورت 10808 آماده می‌کند
5. **مانیتورینگ** — دائماً سلامت کانفیگ‌ها را چک می‌کند
