# LinkedIn Post — huntercensor v1.3.0

---

## English

I'm pleased to announce the release of huntercensor v1.3.0 — a native Windows application for discovering and validating censorship-resistant proxy configurations with a streamlined user experience.

Key improvements in v1.3.0:
- No longer requires administrator rights at startup; elevation is only requested for the specific edge-bypass operation.
- Simplified Home dashboard with core actions: start/stop scan, probe network, discover exit path, and copy or scan QR codes for working configs.
- Comprehensive bilingual user guides (English and Persian) for both normal users and power users.
- Clean UI/UX with advanced controls moved to a dedicated Advanced page.
- Self-contained installer including bundled runtimes and cached configs.

This release is designed for users operating in restricted network environments. The application runs as a regular user, performs background scanning, and allows you to copy or scan a working configuration when ready.

Project repository: https://github.com/bahmany/censorship_hunter

## فارسی (Persian)

با افتخار اعلام می‌کنم نسخه ۱.۳.۰ huntercensor منتشر شد — یک اپلیکیشن بومی ویندوز برای کشف و اعتبارسنجی کانفیگ‌های مقاوم در برابر سانسور با تجربه کاربری ساده و بهینه.

بهبودهای کلیدی در v1.3.0:
- دیگر نیازی به دسترسی ادمین در هنگام اجرا نیست؛ فقط برای عملیات خاص edge-bypass درخواست دسترسی می‌شود.
- داشبورد ساده‌شده Home با اقدامات اصلی: شروع/توقف اسکن، پروب شبکه، کشف مسیر خروج، و کپی یا اسکن QR کانفیگ‌های سالم.
- راهنمای کامل دو زبانه (انگلیسی و فارسی) برای کاربران عادی و حرفه‌ای.
- رابط کاربری تمیز و ساده با تنظیمات پیشرفته منتقل‌شده به صفحه Advanced.
- نصب‌کننده مستقل شامل موتورهای پروکسی و کانفیگ‌های کش‌شده برای استفاده آفلاین.

این نسخه برای کاربرانی طراحی شده که در شبکه‌های محدود فعالیت می‌کنند. برنامه به‌صورت عادی اجرا می‌شود، اسکن پس‌زمینه را انجام می‌دهد و در زمان آمادگی به شما اجازه کپی یا اسکن یک کانفیگ سالم را می‌دهد.
