# راهنمای کاربری huntercensor (فارسی)

![محیط اصلی huntercensor](../Screenshot%202026-03-23%20135206.png)

## 1. huntercensor چیست

`huntercensor` یک برنامه بومی ویندوز است برای پیدا کردن، اعتبارسنجی، کپی کردن و آماده‌سازی کانفیگ‌های مقاوم در برابر سانسور.

این برنامه برای دو گروه طراحی شده است:
- **کاربر عادی** که یک داشبورد ساده و چند عمل اصلی می‌خواهد.
- **کاربر حرفه‌ای** که می‌خواهد به تنظیمات منابع، لاگ‌ها، موتورهای اجرایی، نگهداری و ابزارهای تخصصی دسترسی داشته باشد.

این برنامه عمداً طوری طراحی شده که **برای استفاده روزمره به دسترسی ادمین نیاز نداشته باشد**.

## 2. سیاست دسترسی ادمین

huntercensor برای این موارد **نیازی به ادمین ندارد**:
- اجرای برنامه
- نصب معمولی با installer
- اسکن و جمع‌آوری کانفیگ
- رفرش پس‌زمینه
- اعتبارسنجی کانفیگ‌ها
- کپی کانفیگ‌ها
- نمایش QR Code
- import و export
- مشاهده لاگ‌ها
- probe و discovery شبکه

در ویندوز، فقط عملیات **Execute Bypass** در صفحه `Censorship` ممکن است درخواست دسترسی ادمین بدهد، چون تزریق route یک عملیات سیستمی و سطح‌بالا است.

یعنی:
- برنامه در حالت عادی بدون ادمین اجرا می‌شود
- نصب می‌تواند در سطح کاربر انجام شود
- بیشتر کاربران هیچ‌وقت به ادمین نیاز نخواهند داشت

## 3. صفحه‌های اصلی برنامه

### Home
این صفحه برای بیشتر کاربران کافی است.

در این صفحه می‌توانید:
- اسکن را شروع یا متوقف کنید
- شبکه را probe کنید
- مسیر خروج را پیدا کنید
- همه کانفیگ‌های سالم را کپی کنید
- یک کانفیگ را جداگانه کپی کنید
- یک کانفیگ را در فایل ذخیره کنید
- برای موبایل QR Code بگیرید

### Configs
این صفحه برای زمانی است که می‌خواهید:
- تعداد بیشتری کانفیگ ببینید
- بین Live و Visible فیلتر کنید
- یک کانفیگ یا همه کانفیگ‌های قابل مشاهده را کپی کنید
- فایل import کنید
- دیتابیس فعلی را export بگیرید

### Censorship
این صفحه برای تحلیل و بررسی دقیق است:
- نتیجه probe شبکه
- نتیجه discovery
- مشاهده hop های مسیر
- لاگ خام verification
- اجرای مسیر پیشرفته bypass

### Logs
برای این موارد استفاده می‌شود:
- مشاهده لاگ زنده
- کپی لاگ برای پشتیبانی یا دیباگ
- فیلتر فقط خطاها

### Advanced
این صفحه مخصوص تنظیمات پیچیده است:
- مسیر موتورهای اجرایی
- مدیریت source ها
- تنظیمات Telegram
- عملیات نگهداری و maintenance
- جزئیات پورت‌ها

### About
در این صفحه می‌توانید:
- معرفی پروژه را ببینید
- لینک‌های GitHub را کپی کنید
- مسیر راهنماهای فارسی و انگلیسی را ببینید

## 4. روند عادی استفاده برای بیشتر کاربران

### مرحله 1: شروع اسکن
به صفحه `Home` بروید و روی `Start Scan` بزنید.

اتفاقی که می‌افتد:
- orchestrator شروع به جمع‌آوری کانفیگ‌ها می‌کند
- اعتبارسنجی در پس‌زمینه انجام می‌شود
- کانفیگ‌های سالم به‌مرور ظاهر می‌شوند

### مرحله 2: صبر کنید
این نکته خیلی مهم است:
- گاهی کانفیگ سالم خیلی زود پیدا می‌شود
- گاهی چند **ساعت** زمان لازم است
- در شرایط سخت ممکن است حتی چند **روز** طول بکشد

این برنامه برای کشف طولانی‌مدت طراحی شده است.

### مرحله 3: کپی یا اسکن کانفیگ سالم
وقتی در `Home` کانفیگ‌های سالم ظاهر شدند:
- از `Copy All Working` برای کپی همه کانفیگ‌های سالم استفاده کنید
- از `Copy` برای کپی یک کانفیگ استفاده کنید
- از `Save` برای ذخیره آن در فایل استفاده کنید
- از `QR` برای اسکن با موبایل استفاده کنید

## 5. وارد کردن دستی کانفیگ‌ها

به صفحه `Configs` بروید.

در آنجا می‌توانید:
- از فایل import کنید
- کانفیگ‌ها را دستی paste کنید
- کل دیتابیس را export بگیرید

این بخش زمانی مفید است که از قبل چند کانفیگ دارید و می‌خواهید huntercensor آن‌ها را بررسی و مدیریت کند.

## 6. استفاده از QR Code

در `Home` یا `Configs` روی `QR` بزنید.

در پنجره QR می‌توانید:
- خود QR را ببینید
- URI را کپی کنید
- URI را در فایل ذخیره کنید
- تصویر QR را به‌صورت BMP ذخیره کنید

روند معمول روی موبایل:
- برنامه کلاینت پروکسی را در گوشی باز کنید
- گزینه import by QR را بزنید
- QR روی صفحه کامپیوتر را اسکن کنید

## 7. ابزارهای صفحه Censorship

صفحه `Censorship` برای تحلیل عمیق و رفع مشکل است.

### Probe Network
این عمل وضعیت شبکه را بررسی می‌کند و یک استراتژی پیشنهادی می‌دهد.

### Discover Exit
این عمل تلاش می‌کند اطلاعات gateway، مسیر خروج و context شبکه را پیدا کند.

### Execute Bypass
این عمل پیشرفته است.
در ویندوز ممکن است برای آن درخواست دسترسی ادمین نمایش داده شود، چون route injection نیاز به مجوز سیستمی دارد.

اگر ویندوز اجازه خواست:
- این درخواست فقط برای همان عملیات bypass است
- بقیه استفاده‌های برنامه همچنان بدون ادمین باقی می‌مانند

## 8. نکات نصب

روش پیشنهادی نصب:
- از installer در حالت user-level استفاده کنید
- نصب را در پروفایل کاربر انجام دهید

این برنامه برای سیستم‌هایی طراحی شده که کاربر ممکن است **مجوز نصب ادمین نداشته باشد**.

فایل `setup.bat` هم اکنون در حالت کاربر کار می‌کند.
اگر elevated نباشد، فقط مرحله تنظیم Firewall را رد می‌کند و متوقف نمی‌شود.

## 9. رفع مشکل

### هنوز کانفیگ سالم پیدا نشده
دلایل ممکن:
- اسکن تازه شروع شده
- شبکه خیلی محدود است
- منابع عمومی ضعیف یا قدیمی هستند
- مسیر موتورهای اجرایی اشتباه است

کارهایی که انجام دهید:
- برنامه را مدت بیشتری باز بگذارید
- صفحه `Logs` را بررسی کنید
- مسیر runtime ها را در `Advanced` چک کنید
- در صورت نیاز source های بیشتری اضافه کنید

### کپی کار نمی‌کند
دلایل ممکن:
- هنوز کانفیگ سالمی وجود ندارد
- دسترسی clipboard موقتاً مشکل دارد

کارهایی که انجام دهید:
- پیام toast را ببینید
- صفحه لاگ‌ها را چک کنید
- دوباره تلاش کنید

### QR روی موبایل import نمی‌شود
دلایل ممکن:
- برنامه مقصد از آن نوع URI پشتیبانی نمی‌کند
- تصویر QR واضح نیست
- خود URI توسط کلاینت موبایل قبول نمی‌شود

کارهایی که انجام دهید:
- به‌جای QR از `Copy` استفاده کنید
- URI را در فایل ذخیره کنید
- با یک کلاینت دیگر امتحان کنید

### Execute Bypass درخواست ادمین می‌دهد
این رفتار در ویندوز طبیعی است.
این درخواست فقط برای همان عملیات bypass است.

### Execute Bypass بعد از گرفتن ادمین هم شکست می‌خورد
دلایل ممکن:
- route injection انجام نشده
- gateway یا interface درست resolve نشده
- شبکه probe های verification را block کرده است

کارهایی که انجام دهید:
- لاگ خام edge bypass را در صفحه `Censorship` بخوانید
- discovery را دوباره اجرا کنید
- بررسی کنید که suggested path واقع‌بینانه باشد

## 11. الگوی پیشنهادی استفاده

برای کاربر عادی:
- وارد `Home` شوید
- روی `Start Scan` بزنید
- صبر کنید
- یک کانفیگ سالم را کپی یا اسکن کنید

برای کاربر حرفه‌ای:
- از `Configs`، `Censorship`، `Logs` و `Advanced` استفاده کنید
- source ها و مسیر runtime ها را تنظیم کنید
- خروجی probe و لاگ‌ها را بررسی کنید

## 12. یادآوری نهایی

huntercensor برای کشف واقعی و طولانی‌مدت طراحی شده، نه موفقیت ساختگی و نمایشی.
اگر شرایط شبکه سخت باشد، طبیعی است که پیدا شدن نتیجه خوب زمان‌بر باشد.
