Hunter VPN نسخه 1.1.0 منتشر شد - به‌روزرسانی بزرگ ویندوز!

با افتخار اعلام می‌کنیم Hunter VPN نسخه 1.1.0 با بهبودهای چشمگیر برای کاربران ویندوز منتشر شد!

نکات کلیدی:
- دیالوگ فایل بومی ویندوز برای وارد کردن راحت کانفیگ‌ها
- نصب‌کننده بهینه شده بدون هیچ وابستگی خارجی
- 40% حجم کمتر نصب‌کننده (35.3 مگابایت)
- مدیریت خودکار پردازه‌ها برای آپدیت روان
- سیستم ساخت بهبود یافته با رفع مشکلات Flutter ویندوز

بهبودهای فنی:
- CLI استاتیک لینک شده - بدون نیاز به DLL
- یکپارچگی MSYS2 cmake + ninja + MSVC
- پشتیبانی کامل از Tor با مدیریت پردازه
- پل ارتباطی WebSocket برای وضعیت زنده

Hunter VPN چه کار می‌کند:
- دور زدن سانسور اینترنت با پروکسی دوگانه (SOCKS + HTTP)
- تکنولوژی ضد DPI با تکه‌تکه کردن TLS
- مسیریابی هوشمند با کانفیگ‌های از پیش آماده
- جستجوی خودکار کانفیگ از GitHub و Telegram

دانلود و مستندات:
